Инструкция

Сайт строит графики по точкам из Excel-файла (расширение .xlsx). Для построения графиков загрузите файл, нажав на кнопку подгрузки.

Можно загружать до 5 файлов. Если хотите соединить точки, выберите соответствующий пункт рядом с именем вашего файла.

Пример файла: скачать пример

цвета задаются по умолчанию:

номер файла цвет
1 синий
2 красный
3 зеленый
4 черный
5 серый

в Аналитическом режиме вы можете подгрузить до 3 файлов с точками и сравнить с 1 или двумя функциями по средствам их рисовки

файл или функция цвет
первая функция синий
вторая функция красный
первый файл зеленый
второй файл черный
третий файл серый

Важно!

названия осей берутся из файла для оси x из клетки А1, для оси y из В1. Если файлов несколько, то названия берутся из первого файла
